# 任務評估｜Email 翻譯

> 評估日期：2026-07-08

## 任務描述

收到英文 email 後翻譯成中文

---

## 評估維度

| 維度 | 答案 |
|------|------|
| 頻率 | 200 次/月（極高） |
| 輸出 | 一段翻譯文字，格式不固定 |
| 步驟 | 不太穩定（有時查術語、有時潤稿） |
| 判斷 | 照步驟跑就可以 ✅ |
| 感受 | 充電（做起來開心）⚠️ |
| 價值 | 低價值（做起來沒太大意義）⚠️ |

---

## 建議層級

**Skill**

---

## 理由

| 維度 | 評估 |
|------|------|
| 頻率 200/月 | ✅ 支援 skill（高頻） |
| 步驟不太一致 | ⚠️ 需要設計彈性框架 |
| 照步驟跑就可以 | ✅ 支援 skill |
| 感受：充電 | ⚠️ 但價值低，skill 可釋放時間去做充電的事 |
| 價值：低 | ✅ 適合降級維護 |

---

## Skill 建議內容

| 項目 | 內容 |
|------|------|
| **觸發詞** | 「翻譯 email」、「譯成中文」 |
| **預期輸出** | 一段翻譯文字（格式視乎原文） |
| **步驟初稿** | 1. 讀原文 2. 判斷語氣（正式/休閒）3. 使用 DEEPSEEK 翻譯 4. 對照原文檢查 5. 產出草稿 |
| **第一次驗證方式** | 用真實 email 測試 3 次，確認輸出符合期望 |

---

## 備註

- 這個任務做起來開心，但價值低，建議 skill 化後由 agent 處理
- 釋放出來的時間可以做更高價值的事
