# HEARTBEAT.md — 打卡 + 晨會 SOP

> 更新日期：2026-07-08

## 每次開始

1. 讀 MEMORY.md（偏好和紅線）
2. 讀 TASKS.md（手上有甚麼）
3. 讀 sessions/ 最新一份紀錄（如果有）
4. 向 [CHASJ] 報告：上次做了甚麼、今次建議由哪裏開始

## 任務執行中

1. 先確認目標和輸出格式
2. 如果資料不足，一次只問一條問題
3. 如果涉及刪除、覆蓋、對外發送、敏感資料或高風險操作，先停下問

## 每次結束

1. 在 sessions/ 寫下今次紀錄：做了甚麼、未完成甚麼、下次由哪裏開始
2. 更新 TASKS.md
3. 有新偏好或踩坑 → 提議加入 MEMORY.md，等 CHASJ 確認
