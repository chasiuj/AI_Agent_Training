# HEARTBEAT.md — 打卡 + 晨會 SOP

> 更新日期：2026-07-08

## 每次對話開始（Start Routine）

1. **讀 LATEST_STATE.md**
   - 了解上次做到哪裏
   - 下次計劃由哪裏開始

2. **讀 TASKS.md**
   - 手上有甚麼進行中的任務
   - 待辦任務有哪些

3. **讀 MEMORY.md**
   - 有甚麼偏好要套用
   - 有甚麼紅線不能踩

4. **向我報告（三句以內）**
   - 上次完成了甚麼
   - 今次建議由哪裏開始
   - 有沒有等待我決定的事

5. **問我：「今次的主力是甚麼？」**

---

## 任務執行中

1. 先確認目標和輸出格式
2. 如果資料不足，一次只問一條問題
3. 如果涉及刪除、覆蓋、對外發送、敏感資料或高風險操作，先停下問

## 每次結束

1. 在 sessions/ 寫下今次紀錄：做了甚麼、未完成甚麼、下次由哪裏開始
2. 更新 TASKS.md
3. 有新偏好或踩坑 → 提議加入 MEMORY.md，等 CHASJ 確認

---

## 規則

- 如果看不到以上 files，請叫我貼上內容或提供 file list，不要假裝看過
- 報告要短。不要把整份 file 讀出來，只講重點
- 未得到 CHASJ 確認前，不要更新任何 file
